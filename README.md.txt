Данный сайт содержит информацию о техниках обучения и образоавании в целом. 
Он реализован на ванильном css и html без подключения Java Script и дополненительных библиотек.
Классы всех элементов создавались на основе методологии БЭМ, файловая структура организована по БЭМ: Nested.

Это одностраничный сайт, содержащий ссылки на другие ресурсы. 
Все сылки при наведении мыши плавно становятся прозрачными, 
это реализовано при помощи таких css свойств как transition и opacity.
Сайт содержит анимированные элементы, движение которых прописано через css свойства animation и кейфреймы.
Верстка и размещение элементов на странице реализовано преимущественно через флекс свойства и таблицы.
В блоке "Видео на TED" находятся два видео-фрейма, взятых с сайта YouTube.
